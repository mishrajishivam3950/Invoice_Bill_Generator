# Invoice Generator - ReactJs

![React](https://img.shields.io/badge/React-18181b?style=for-the-badge&logo=react&logoColor=61DAFB)
![TailwindCSS](https://img.shields.io/badge/TailwindCSS-06B6D4?style=for-the-badge&logo=tailwind-css&logoColor=white)

This Invoice Generator project, built with React and TailwindCSS, allows users to add items, quantities, prices, tax rates, and discounts. It enables the download of generated invoices as PDFs for printing. The project uses html-to-image to capture data from the modal, converting it from a canvas to a PDF using jsPDF.
